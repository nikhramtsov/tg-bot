# tg-bot

### Запуск:
```bash
docker-compose up --build -d
```

### Технологии и библиотеки

- Python 3.12
- Postgres 15
- Docker
- Aiogram 3
- SQLAlchemy 2
- Alembic
- aiocron

Полный список в requirements.txt 

Релизованы "регистрация" командой /start, эхо, добавление и просмотр заказа, рассылка по времени.
